# Pi Network Configuration Tool

Automated network configuration management for Raspberry Pi/Debian systems.

## Overview

Tool manages WiFi connectivity with automatic fallback to access point mode when no connection available.

## Project Structure

```
pi-netconfig/
├── governance/     # Operational rules
├── workspace/      # Execution space
├── docs/           # Technical documents
├── src/            # Source code
└── deprecated/     # Archive
```
